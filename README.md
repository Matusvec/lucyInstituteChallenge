# lucyInstituteChallenge

## Setup

### Datasets
The datasets are not included in this repository due to their size (~2GB). 

**To get the datasets:**
1. Download the `Datasets/` folder from [OneDrive link - replace with your actual link]
2. Place it in the project root directory

Your folder structure should look like:
```
lucyInstituteChallenge/
├── Datasets/
│   ├── Environmental Health Dataset/
│   ├── IQVIA/
│   ├── Malaria/
│   ├── MalariaDatasetsCleaned/
│   └── SupplementalDatasets/
├── README.md
└── ...
```
