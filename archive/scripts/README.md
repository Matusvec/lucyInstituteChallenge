# Scripts

Standalone runnable scripts (not imported by `main.py`).

## scripts/python/

| Script | Purpose | Run |
|--------|---------|-----|
| `_test_chart.py` | Save prescriptionsVsOverdose chart as PNG | `python scripts/python/_test_chart.py` |
| `seg_bar_graph_rough.py` | Stacked bar: prescriptions by year | `python scripts/python/seg_bar_graph_rough.py` |

Run from **project root**.
