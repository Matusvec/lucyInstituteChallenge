# Data Flow & Visualization Dependencies

## Active Visualizations (in main.py)

| Command | Visualization | Data Source | Queries Used |
|---------|---------------|-------------|--------------|
| `map-illicit` | State-level illicit overdose spread (1999ŌĆō2018) | `output/cdc/cdc_illicit_overdose_by_state_year.csv` | `cdc-drug` (load_wonder_drug_types) |
| `map-county` | County overdose death rate (2008ŌĆō2017) | `Datasets/cdc/overdose_by_county_year_2008-2017.csv` | None (CDC CSV only) |
| `map-fentanyl` | County fentanyl spread (2008ŌĆō2017) | `Datasets/cdc/overdose_by_county_drugtype_2008-2012.csv`, `_2013-2017.csv` | None (CDC CSV only) |
| `map-dashboard` | Multi-metric county map (IQVIA + CDC) | `output/county/iqvia_county_year_panel.csv` + CDC county CSVs | `county` (county_panel) |
| `chart-heroin-fentanyl` | Heroin vs Fentanyl bar chart (1999ŌĆō2018) | `output/cdc/heroin_vs_fentanyl_1999-2018.csv` | Pre-generated (in repo) |

## Map Data Dependencies

| Map | Input Files | Generated By |
|-----|-------------|--------------|
| `illicit_overdose_spread_map.html` | `output/cdc/cdc_illicit_overdose_by_state_year.csv` | `python main.py cdc-drug` |
| `county_overdose_spread_map.html` | `Datasets/cdc/overdose_by_county_year_2008-2017.csv` | Raw CDC data (in repo) |
| `fentanyl_spread_map.html` | `Datasets/cdc/overdose_by_county_drugtype_2008-2012.csv`, `_2013-2017.csv` | Raw CDC data (in repo) |
| `county_dashboard_map.html` | `output/county/iqvia_county_year_panel.csv` + CDC county CSVs | `python main.py county` (needs DB) |

## No Database Required

These commands run without a database connection (read from CSV only):

- `map-illicit` (if `cdc-drug` was run previously)
- `map-county`
- `map-fentanyl`
- `map-dashboard` (if `county` was run previously)
- `chart-heroin-fentanyl`
